# 芝山 恵生（Kei Shibayama）

機械工学専攻学生。  
工学と社会の関係を問いながら、宇宙工学分野を中心に学んでいます。

技術を単なる手段としてではなく、  
どのような前提で設計され、どのように社会の中で機能するのかに関心があります。

現在は特に、人工衛星の制御に関する研究分野への進学を目標としています。

---

## 研究関心

- 人工衛星の制御・運用
- 宇宙工学と社会課題の接続
- 技術と意思決定の関係
- 国際協働による工学的課題解決

---

## 関心の背景

これまで、宇宙分野の教育プログラムや国際的な工学ワークショップに参加する中で、  
技術は知識や設計能力だけで成立するものではなく、  
異なる前提や価値観を持つ人々の間でどのように課題が定義され、  
どのように判断されるかによって形づくられることを実感してきました。

そのため現在は、技術そのものだけでなく、  
技術が成立する過程や環境にも関心を持っています。

---

## 現在の関心領域

- 宇宙工学分野の研究環境の理解
- 国際的な工学教育の比較
- 研究プロセスの観察と記録
- 技術の社会実装過程の分析
